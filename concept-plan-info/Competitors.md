# Конкурентний аналіз — Сама Ліпила (концепт-редизайн)

> **Дата:** 12.06.2026
> **Ніша:** заморожені напівфабрикати власного виробництва, локальні бренди.
> **Аналізовано:** 4 конкуренти — 3 українських + 1 чеський (для UX-референсу).
> **Мета аналізу:** виявити стандарт ніші (must-have), точки диференціації та сліпі зони для редизайну.

## 1. Місиво

**URL:** [misyvo.com](https://misyvo.com/)
**Оцінка:** 4.5/10

### Структура головної
- **Header:** лого, кнопки навігації (Продукція / Як нас знайти / Магазин), кнопки користувача (Увійти / Кошик).
- **Hero:** 2 логотипи + 2 CTA-кнопки («Замовити в один клік», «Наші магазини»).
- **Інфо-секція:** Доставка / Оформлення / Вартість.
- **Каталог власного виробництва:** з фільтрами по категоріях.
- **Каталог магазину:** з фільтрами по категоріях.
- **Інструкція замовлення:** 3 кроки (вибрати продукцію → зв'язатись із менеджером → підготовка та доставка кур'єром). Інформація про мінімальне замовлення та графік доставки.
- **Локації:** адреса + інтерактивна мапа.
- **Footer:** Політика конфіденційності, Сертифікати (не клікабельні).
- Кнопка «Догори».

### Структура магазину
- Header дефолтний.
- Каталог із сортуванням (популярність / ціна / за замовчуванням / останні).
- Aside bar із фільтрами по категоріях.
- Footer дефолтний.

### Структура кошика
- Таблиця: Товар — Ціна — Кількість — Проміжний підсумок.
- Кнопки: «Продовжити покупки», «Оновити кошик».
- Підсумки: проміжний підсумок + загалом + кнопка «Перейти до оформлення».

### Що добре
- Дизайн лого.
- Загальна естетика сторінки.
- Картинки з головним персонажем — викликають довіру.
- Особистий кабінет.

### Що погано
- **Брехня про замовлення «в один клік»** — кнопка не виконує те, що обіцяє.
- Однакові зображення для 13 з 14 видів вареників.
- На головній за замовчуванням 14 видів вареників — складається враження, що продають лише вареники.
- Магазин на головній із різностильовими товарами.
- Дивна категоризація магазину (чіпси поряд із оливками й консервами).
- Не клікабельні «Політика конфіденційності» та «Наші сертифікати» у Footer.
- Хаотична стилістика фото в каталозі магазину (телефонні / студійні / стокові).

## 2. якVdoma

**URL:** [yakvdoma.com](https://yakvdoma.com/)
**Оцінка:** 6.5/10

### Структура головної
- **Header:** лого + назва «Як вдома» + підназва «Магазин виробництва». Справа — кошик, ЛК, burger-меню.
- **Навігація:** Меню / Зони доставки / Залишити відгук.
- **Photo header** без тексту.
- Одразу під ним — каталог: зліва фільтри по категоріях, посередині товари (за замовчуванням починаючи з пельменів, довгий скрол).
- Справа — інфо-блок: робочий час, адреса, телефон, кнопка «Надіслати відгук».
- **Footer:** контакти (адреса, телефон, email, чат), соцмережі, інтерактивна карта, кнопка «Отримати розташування» (Google Maps). Cookie policy + Умови користування + Політика конфіденційності — все клікабельне.

### Картка товару
- Назва, кількість штук, ціна, склад, кнопка «Показати більше», кнопка «Додати», фото, вага, **кількість лайків**, кнопка «Поділитись».

### Окремі сторінки
- **Зони доставки** (без header/footer): інтерактивна мапа з обмеженими чорним зонами, права колонка з умовами (в межах міста — від 700 грн безкоштовно; за межами — від 1500 грн).
- **Надіслати відгук:** оцінка по 5-бальній системі зірочками, дві категорії (Страви, Сервіс), необов'язкові контакти (телефон, email, ім'я).
- **Реєстрація:** Google / Apple / email. Переваги вказані одразу: швидше оформлення, історія замовлень. Реєстрація через Google — ~10 секунд.
- **Burger-меню:** ЛК, контакти, адреса, телефон, email, посилання на сайт, соцмережі, «Корисна інформація».

### Структура кошика
- Стандартна. Додавання / видалення / зміна кількості з автоматичним перерахунком ціни.
- **Поле для коментаря закладу.**

### Що добре
- Зони доставки на інтерактивній мапі окремою сторінкою.
- Лайки на товарах + лічильник у «Популярному» (соціальний сигнал).
- Cookie / Умови / Політика конфіденційності — все клікабельне.
- Категорія «Популярне» (по кількості лайків).
- Пошук продуктів по назві.
- Єдина стилістика фото.
- Відгук можна залишити лише оцінкою, без обов'язкового тексту.

### Що погано
- **Одразу на головній — каталог, без жодного введення.** Перша позиція — страва з білого амура (сире фото на тарілці), що відштовхує нового відвідувача.
- Повторювані фото на різних продуктах однієї категорії.
- Footer недосяжний — щоб до нього дістатися, треба проскролити весь каталог.
- Корисна інформація (алергени, про компанію, політики) захована в burger-меню.

## 3. Добреїсти

**Оцінка:** 7.5–8/10

### Структура головної
- **Header:** лого зліва + навігація (Про нас / Меню / Доставка / Контакти / Співпраця) + CTA «Замовити». Натискання на «Контакти» — скрол у Footer.
- **Hero:** автоматично перемикаючі карусель-картки (логотип + короткий текст на фоні їжі, 2 картки товарів).
- **Секція категорій:** котлетки, вареники, пельмені, сирнички, налисники, голубці, сезонні страви. **Клікабельні тільки фото** — UX-проблема (не видно, що з елементами можна взаємодіяти).
- **«Як зробити замовлення»** — інтерактивні пункти, з'являються при скролі.
- **«Про нас»:** великий логотип, фото персоналу, інфо про компанію, соцмережі.
- **B2B-секція:** кнопки «Дізнатись більше» + «Залишити заявку».
- **Footer:** лого, Visa/Mastercard, Контакти, Адреси, адреси фудмаркетів, Політика конфіденційності (клікабельна).

### Структура меню
- Header + меню з категоріями + пошук.
- 20 карток за замовчуванням на сторінці. Перемикач кількості: 20 / 50 / 100.
- Картка: фото, назва, вага, ціна, кнопка «В кошик».
- Стиль фото єдиний: «домашнє, тарілка на рушничку, соковите».

### Структура Доставки
- Інформація про час і умови.
- Дві картки «Карта» / «Готівка» з логотипами.

### Структура Співпраці (B2B)
- Hero: заголовок «Поставки домашніх напівфабрикатів для бізнесу у Львові» + підтекст + CTA «Отримати пропозицію».
- Деталі що отримує клієнт після заявки.
- Accordion (`<details>/<summary>`): прайс, умови співпраці, асортимент, консультації.
- Інформація про асортимент + кнопка «Переглянути меню».
- **Цифри:** кількість найменувань, кількість постійних клієнтів, тонн продукції на тиждень.
- Локації щоденної доставки.
- Переваги співпраці.
- Як почати співпрацю.
- Секція «Потрібна швидка відповідь» з телефоном і email.

### Структура кошика
- Стандартна.
- Поля: ім'я, телефон, спосіб зв'язку (Telegram / Viber / дзвінок), промокод, коментар, метод доставки.
- Оплата: готівка (з коментарем про решту) або картка.
- На головній кошик — лише як sticky-логотип.

### Що добре
- **Переконлива B2B-сторінка**, інформаційна, з цифрами і вигодами.
- Грамотна секція Hero — одразу викликає апетит (2 картки товарів + інтерактивна карусель).
- Поділ товарів на категорії на головній, а не одразу каталог.
- Єдина стилістика фото товарів.
- Анімації при скролі.
- Зручне оформлення замовлення + вибір каналу зв'язку.
- Промокоди.

### Що погано
- У секції категорій клікабельні **лише фото** — не очевидно, що з елементами можна взаємодіяти.
- Занадто висока секція Hero — багато скролу до наступної секції.
- Забагато анімацій (кожна секція має свою).
- Контакти лише у Footer, без інтерактивної мапи.
- Відсутність особистого кабінету.

## 4. multicook

**URL:** [multicook.cz](https://www.multicook.cz/)
**Оцінка:** не виставлена (іноземний референс для UX, не пряма ніша)

### Структура головної
- **Header:** Головна / Замовити онлайн / Рецепти приготування / Про нас / Контакти / Співпраця + лого + ЛК + кошик (з кількістю замовлень). Велика кнопка «Рецепти приготування».
- **Hero:** фото пельменів + CTA для замовлення онлайн.
- **Відео приготування** пельменів (фарш, розкатка тіста).
- Секція «Рецепти приготування» + найпопулярніші страви.
- Секція «Відкрийте смак нових можливостей» + кнопка «Спробуйте новини» — **не клікається**.
- Секція «Наші переваги»: якість і свіжість, можливість замовити повні обіди. Меню стилізоване як ресторанне, картинки клікабельні.
- Підписка на email-розсилку.
- Заохочувальна секція з фото і текстом.
- Інтерактивна мапа з робочими годинами і адресою.
- **Footer:** соцмережі (іконки) + 3 секції (Де нас знайти, Контакти, Кар'єра — з можливістю надіслати резюме email).

### Випадаючий блок «Замовити онлайн»
При наведенні в header — окремий блок із категоріями (вареники, пельмені, сирники, чебуреки). Перехід у каталог з ціною, фото, кнопкою додавання в кошик.

### Рецепти приготування
- Голубці з фаршем, фрикадельки, пельмені трьох видів м'яса.
- На кожному рецепті — лайки + кількість переглядів.

### Сторінка «Про нас»
Майже нічого нового — дуже схожа на головну.

### Сторінка «Співпраця»
- Окрема анкета.
- Видно персонал, текст від власника.
- Невеликий блок із відео (готування фрикадельок).
- Інтерактивна мапа.

### Реєстрація
Через Google — швидка.

### Що добре
- Наявність розділу «Рецепти приготування» (content marketing).
- Випадаючий блок із категоріями при наведенні на «Замовити онлайн».
- Окрема сторінка про співпрацю.
- **Відео приготування на головній** — підвищує апетит і довіру.
- Фільтр у каталозі по ціні.
- Окремий фільтр з акціями.

### Що погано
- Контакти лише у Footer.
- Невелика кількість рецептів.
- Кнопка «Спробуйте новини» не клікається.
- «Про нас» майже не відрізняється від головної.
