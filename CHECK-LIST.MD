4. Картинки 
alt у всех <img> и <Image>, осмысленный (не "image1", не "фото")
декоративные картинки: alt=""
все фото WebP, сжаты (≤300 КБ для hero, ≤150 КБ для контентных)  лого/иконки ≤30 КБ
<Image> из astro:assets везде, где можно (авто width/height/decoding)
<Picture> с widths={[400,800,1200]} для крупных адаптивных
background-image: только декор, не контент; помнить что не оптимизируется
loading="lazy" на всём, что ниже первого экрана / loading="eager" + fetchpriority="high" на hero-картинке
